# 01. 研究の前提と背景

## 1.1 研究者のコンテキスト

- Deloitte Tohmatsu にて内部監査AI領域に従事
- 監査エージェントシステムの開発経験（Agentic RAG → Deep/Normal/Simple エージェント → ワークペーパー自動転記の3フェーズパイプライン）
- LLMアーキテクチャ設計、因果推論は初学者段階
- 目標: 内部統制・因果推論・GenAIを組み合わせた研究テーマの確立

## 1.2 研究の出発点

内部統制、因果推論、GenAIを組み合わせた研究テーマを多角的に検討した結果、以下の4つの方向性を特定した。

| 方向 | テーマ | 新規性 | リスク |
|------|-------|-------|-------|
| A | LLMエージェントの判断信頼性を因果推論で評価 | 中（先行研究に追いつかれる可能性） | 中 |
| B | 内部統制の有効性をLLM＋因果推論で評価 | 中〜高（LLMによる変数抽出が差別化） | 中 |
| C | A+Bの統合フレームワーク | 高 | スコープ過大 |
| **D** | **LLM環境ツイン（OCT）による因果推論** | **極めて高い** | **理論的正当化が課題** |

**方向Dを主軸として追求する。** 他の方向は必要に応じて副次的に取り込む。

## 1.3 方向Dに至った着想の経緯

1. 従来の因果推論はDAGによる事前定式化を要求するが、内部統制のような複雑な環境では定式化できない変数（組織文化、暗黙知、モチベーション等）が多い
2. LLMエージェント研究は「人間の業務再現」が主流だが、「環境そのもの」をシミュレートするデジタルツインとして使えるのではないか
3. ツインへの介入（do演算）により、事前にDAGを定義しなくても因果関係を探索・検出できる
4. これは因果推論のパラダイム転換の提案となりうる

## 1.4 既存の監査エージェントパイプラインとの関係

本研究は既存の監査エージェントパイプラインとは独立した研究である。ただし、OCTで発見された因果関係を既存パイプラインの改善に活用する接続パスは将来的にありうる。

```
既存パイプライン:
  Phase ① Agentic RAG → 監査手続生成
  Phase ② Deep/Normal/Simple Agent → 実行
  Phase ③ ReAct Agent → ワークペーパー転記

OCTの位置づけ:
  既存パイプラインの「上流」に位置する。
  「どの統制に監査資源を集中すべきか」の判断材料を提供する。
  パイプラインの構成要素ではなく、独立した分析ツール。
```
